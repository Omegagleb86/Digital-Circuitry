## Декодеры

>[!info] Правила
>- Во всех многоразрядных устройствах, кроме регистрах, чем меньше индекс, тем младше разряд
>- Прямые выводы (входы или выходы) имеют активного сигнала логической еденицы, а инверсный логического нуля

Для декодеров с прямыми выходами:
$$F=\vee Q_H=\overline{\vee Q_L}$$
$Q_H\ -$ Выводы на которых функция принимает логическую единицу
$Q_L\ -$ Выводы на которых функция принимает логического нуля

Для декодера с инверсными выходами:
$$F=\overline{\wedge Q_H}=\wedge Q_L$$
Для реализации функции на декодерах

## Карта Веча для функции
1. Составить карту Веча функции
2. Составить маскирующую матрицу выбранного порядка мультиплексора
   (желательно чтобы положение прямого значения аргумента прямого значения адресного входа, в маскирующей матрице)
3. Наложить матрицу на карту
   В результате положения определяет какие аргументы функции наложить на какие адресные входы, затем минимизировать в каждой области маскирующюю матрицу отдельно, а результат минимизации записывается по тем аргументам, каждые не наложить на адресные входы мультиплексора и...
4. Строим схему

$$f=\overline{a}\overline{b}c\vee\overline{a}\overline{b}\overline{c}\vee\overline{a}b\overline{c}\vee abc$$
