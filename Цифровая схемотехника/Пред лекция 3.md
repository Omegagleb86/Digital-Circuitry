# Мультиплексоры
Мультиплексоры $\equiv$ multiplexer
Селектор данных $\equiv$ data select
Коммутирующие (переключающие функции в ЦСТ)
Он способен подключить один из нескольких информирующих входов к единственному выходу

![[Цифровая схемотехника/Цифровая схемотехника.excalidraw.md#^group=Lw5-yuvIsDjRIXIjCjusT]]
- $D$ (data) $-$ Информационные входы
- $A$ (address) $-$ Адресные входы
- $S$ (strobe) $-$ стробирующий вход
  strobe $-$ разрешающий принимать и обрабатывать сигналы, поступающие на другие входы
- $Q$(Quit) $-$ выход

Мультиплексоры входят в состав многофункциональных цифровых устройств
Микросхемы любых коммутирующих устройств имеют в 3 части маркировки 1 букву К, а во второй части к какой группе
Мультиплексоры относятся к группе =='прочее'==
Чаще всего мультиплексоры называют селекторы-мультиплексоры (или мультиплексоры-селекторы)
Поэтому вместо $MUX$ используют $MS$

Если при активном стробирующем входе в адресе подать двоичный код $(00; 01; 10; 11)$ номера входа $D0;\ D1;\ D2;\ D3$, то указанный вход соединяется с выходом, остальные же сигналы на вход не проходят

![[Цифровая схемотехника/Цифровая схемотехника.excalidraw.md#^group=U8F1X7o7bSL1MYrVtpvqN]]
$10_2=2$: Подключён $D2$
Мультиплексоры способные пропускать только 1 сигнал называются одноразрядными
![[Цифровая схемотехника/Цифровая схемотехника.excalidraw.md#^group=kgnT8nwr]]
Если подать на разрешающий вход активный 0, а в $AB$ подать двоичный 0, то $1C0$ и $2C0$ будет передана в $1Y$ и в $2Y$

>[!info] Разновидности схем
>```КР1533КП19``` $-$ инвертный вход
>```КР1533КП12``` $-$ буферизация
>```КР1533КП17``` $-$ буферная инвертная


# Демультиплексоры
Демультиплексоры (demultiplexure) коммутирующие функции, способен подключить единственный вход к одному из нескольких выходов
## Демультиплексор $1\times4$
![[Цифровая схемотехника/Цифровая схемотехника.excalidraw.md#^group=lbU5Jqh0koIv_rpyj5BKY]]
Принцип работы демультиплексора похож на принцип работы мультиплексора, если на стробирующий вход пожать активный 0, а на адрес подать двоичный код номера выхода, то данный выход будет передавать данные из D, на остальных уровнях будет пассивная 1
ТТЛШ - в одну сторону
КМОП - в обе стороны как мультиплексор так и Демультиплексор
Микросхемы любой структуры

В качестве демультиплексора можно использовать дешифратор, имеющий хотя бы 1 разрешающий вход

Если дешифратор имеющий больше 1 разрешающего входа, то один разрешающий является информационным, а все остальные являются стробирующем. Входы для входов используются в качестве адресных.
